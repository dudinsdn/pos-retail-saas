# POS Retail SaaS

A modern Retail Point of Sale (POS) system built with Rust using Clean Architecture.

---

## Features

- Clean Architecture
- REST API
- PostgreSQL
- SQLx
- Axum
- JWT Authentication (Planned)
- Inventory Management (Planned)
- Purchase Management (Planned)
- Sales Management (Planned)

---

## Tech Stack

- PostgreSQL
